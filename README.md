# ExpertBankCli
Client pour ExpertBank (https://github.com/SteveHoareau18/ExpertBank)


# Installation dépendences NodeJS
```
npm i
```

# Build NodeJS (en dev)
```
npm run watch
```

# Installation dépendances Composer
```
composer i
```

# Démarrer le serveur (en dev)
```
symfony server:start
```
